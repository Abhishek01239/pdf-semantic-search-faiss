# PDF Semantic Search Engine using FAISS

A mini RAG-style retrieval system built with Python, Sentence Transformers, and FAISS.

## Features

- PDF Text Extraction
- Text Chunking
- Embedding Generation
- Semantic Search
- FAISS Vector Indexing

## Tech Stack

- Python
- PyPDF2
- LangChain Text Splitters
- Sentence Transformers
- FAISS

## Project Flow

PDF → Chunks → Embeddings → FAISS → Semantic Search
